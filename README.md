# formula-one-analysis
This project presents an exploratory data analysis of Formula 1 race results from 2021 to 2024.
The objective is to analyze team and driver performance trends using statistical aggregation, visualization, and dimensionality reduction techniques.

The analysis focuses on:

Team dominance and point accumulation

Seasonal performance progression

Starting grid vs final position relationships

Distribution of race wins

Fastest lap performance trends

Correlation between numerical race metrics

Dimensional pattern analysis using PCA
